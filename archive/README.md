# Archive Directory Contents

## Debug Files (`archive/debug/`)
Debug files and temporary output from development:
- `debug_*.py` - Debug scripts
- `output.txt`, `schedule_output*.txt` - Temporary output files
- `debug_page.html` - Debug HTML pages

## Analysis Files (`archive/analysis/`)
Analysis scripts and historical data from development:
- `analyze_*.py` - Various analysis scripts
- `compare_*.py` - Comparison utilities
- `check_*.py` - Validation scripts
- `test_*.py` - Old test files (except `test_all_weeks.py` which is active)
- `week6_analysis*.txt` - Week 6 analysis outputs
-`advanced_experiments.txt` - Experiment notes
- `*_schedules.csv` - Old schedule exports

## Active Scripts (Root Directory)
Core scripts that are actively used:
- `test_all_weeks.py` - Main test runner
- `generate_schedule.py` - Schedule generation
- `gui_web.py` - Web interface
- `migrate_activity_names.py` - Data migration utility
- `constrained_scheduler.py` - Core scheduler logic
- `models.py`, `activities.py`, `io_handler.py` - Core modules
